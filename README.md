# Lappa_Api

## Description
Magento 1 REST API Lappa Api

## Plugin Installation

Download archive. Create a folder Lappa on ```/app/code/community``` . In the Lappa folder, paste the folder ```Interfacing```
Folder ```etc``` file ```Lovat_Interfacing.xml``` paste in the folder behind the path ```/app/etc/modules```.
Reset the cache on the site ```System->Cache Management```
